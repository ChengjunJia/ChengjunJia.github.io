Software Requirement for Production

  • MS Office(Onenote and Outlook are necessary); WPS; Typora;
  • Youdao Note;
  • Youdao Dict;
  • Everything: 很好用的搜索软件
  • 火柴官网: 原来的火萤酱, 用于快速打开应用或文件
  • WGesture: 鼠标手势
  • Bandzip, 7-zip: 极好用的压缩软件
  • Chrome, Firefox
  • Anaconda for Programming
  • MobaXterm for SSH
  • Snipaste: 截图软件, 快捷键F1直接截图, F2贴图; 试一下你就会爱上它
  • SumPDF for Reading
  • XShell; Winscp/Filezilla; Putty
  • 卡硬：检测笔记本硬件性能的集合软件
  • 盘姬工作箱

Website

  • Awesome Windows, https://amazing-apps.gitbook.io/windows-apps-that-amaze-us/zh-cn/security
  • Crack Softwares in China: (zdfans.com)[http://zdfans.com]
  • Inoreader: 使用RSS查看各种新闻实事
  • 虫部落：巨好用的搜索集合网站
  • Alternative: 通过该网站找到软件的替代品
  • weavesilk(绘制科幻图像的有趣的网站): weavesilk.com
  • Latex: 表格绘制: http://www.tablesgenerator.com/; 在线LaTeX编辑和编译器: overflow.com; 在线LaTeX公式: https://www.codecogs.com/latex/eqneditor.php?lang=zh-cn
